HORSE EVALUATION INTAKE INSTRUCTIONS

INTERNAL USE / APPLICATION BUILD REFERENCE

Purpose: This document explains the use of the Horse Evaluation Client Intake Form and how information flows into evaluation engagements.

WHEN TO PRESENT THIS FORM

Present when:

☐ Client requests assessment of a specific horse

☐ Client is considering purchase or lease

☐ Client wants suitability feedback

☐ Client needs professional observation before proceeding

PRIMARY INFORMATION COLLECTED

The form establishes:

☑ Client identity

☑ Horse identity

☑ Horse location

☑ Intended purpose

☑ Rider requirements

☑ Evaluation expectations

REQUIRED DATABASE FIELDS

Required:

Client Information

Horse Information

Horse Location

Evaluation Purpose

Intended Use

CONTRACT GENERATION INPUTS

Used for:

Horse Evaluation Agreement

Populates:

☑ Client information

☑ Horse information

☑ Evaluation scope

☑ Intended use

☑ Disclosures

INTERNAL REVIEW ITEMS

Before accepting engagement:

Confirm:

☐ Horse location is available

☐ Scope is clear

☐ Client understands evaluation limitations

☐ Veterinary services are not being represented unless separately provided

APPLICATION DISPLAY NOTES

Keep the form focused.

Do not overwhelm the client with:

medical history

competition history

registration details

unless relevant to the requested evaluation.

COMMON ERRORS TO AVOID

Do not:

☐ Guarantee soundness

☐ Guarantee performance

☐ Replace veterinary examination

☐ Present evaluation as a professional certification
