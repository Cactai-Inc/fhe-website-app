HORSE SALE INTAKE INSTRUCTIONS

INTERNAL USE / APPLICATION BUILD REFERENCE

Purpose: This document explains how the Horse Sale Client Intake Form should be used to create a seller engagement and prepare transaction services.

WHEN TO PRESENT THIS FORM

Present when:

☐ Owner wants assistance selling a horse

☐ Owner wants help marketing a horse

☐ Owner needs assistance screening buyers

☐ Owner requests transaction support

Do not use when:

☐ Client is only seeking an evaluation

☐ Client is looking to purchase a horse

PRIMARY INFORMATION COLLECTED

The form establishes:

☑ Seller identity

☑ Horse ownership information

☑ Horse profile

☑ Sale objectives

☑ Requested assistance

☑ Disclosure information

REQUIRED DATABASE FIELDS

Required before engagement:

Seller Information

Horse Information

Ownership Information

Sale Objective

Requested Services

CONTRACT GENERATION INPUTS

Used for:

Seller Representation Agreement

Horse Sale Agreement

Populates:

☑ Seller information

☑ Horse information

☑ Sale parameters

☑ Scope of services

☑ Compensation terms

INTERNAL REVIEW ITEMS

Confirm:

☐ Seller is authorized to sell horse

☐ Ownership information is available

☐ Known issues have been disclosed

☐ Marketing expectations are realistic

APPLICATION DISPLAY NOTES

Use conditional questions.

Example:

If seller selects “transaction assistance only”:

Skip marketing questions.

If seller selects “full sale assistance”:

Display marketing and buyer screening sections.

COMMON ERRORS TO AVOID

Do not:

☐ Represent ownership without documentation

☐ Promise a sale timeline

☐ Guarantee buyer interest

☐ Omit known disclosures
